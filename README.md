# Next Android 库总览

NextAndroid 是一个集成Android App开发工具库

## Click Proxy - 点击代理

基于NextEvents的扩展，一个@ClickEvt注解即快速实现绑定UI组件的点击处理处理。

## Inject View - 自动注入View

View自动注入处理，不再findViewById。
